## Accessibility

NumberInput implements the [Spinbutton design pattern, per WAI-ARIA 1.1](https://www.w3.org/TR/wai-aria-practices-1.1/#spinbutton), with some additional work to ensure that assistive technology will announce value changes whenever an increment or decrement button is pressed.

### Labelling

A NumberInput can be labelled using a FieldLabel, `aria-label` or `aria-labelledby`.

#### Using FieldLabel

Since a NumberInput is an input group containing the Textfield, increment and decrement Buttons, when labelling using a FieldLabel, use implicit labelling or add the `aria-labelledby` prop to explicitly associate the group of controls to the FieldLabel.

##### Implicit labelling will add `aria-labelledby` automatically

<FieldLabel label="Enter a value">
  <NumberInput min={0} max={100} value={0} />
</FieldLabel>

```jsx
<FieldLabel label="Enter a value">
  <NumberInput min={0} max={100} value={0} />
</FieldLabel>
```

##### Explicit labelling requires `aria-labelledby` be added manually

<FieldLabel label="Enter a value" id="numberinput-label-id" labelFor="numberinput-id" />
<NumberInput
  id="numberinput-id"
  aria-labelledby="numberinput-label-id"
  min={0}
  max={100}
  value={0} />

```jsx
<FieldLabel label="Enter a value" id="numberinput-label-id" labelFor="numberinput-id" />
<NumberInput
  id="numberinput-id"
  aria-labelledby="numberinput-label-id"
  min={0}
  max={100}
  value={0} />
```
