## Overview

A **Calendar** handles date selection.

## StartDay

This prop can have value between 0 and 6. Default is 0.

<div style={{width: '280px'}}>
  <Calendar startDay={1} value="today" />
</div>

```jsx
<Calendar startDay={1} value="today" />
```

## Controlled Calendar

The calendar has defualt value set to today

<div style={{width: '280px'}}>
  <Calendar defaultValue="today" />
</div>

```jsx

```

## Single and Range selector Calendar

The calendar supports single date selection and range selection as well. Default is single and 'selectionType' prop can be used for setting the date range.

### Range

<div style={{width: '280px'}}>
  <Calendar selectionType="range" />
</div>

```jsx
<Calendar selectionType="range" />
```

### Range with value

<div style={{width: '280px'}}>
  <Calendar selectionType="range" value={['2015-01-15','2015-01-19']} />
</div>

```jsx
<Calendar selectionType="range" value={['2015-01-15','2015-01-19']} />
```

## State

Like other props , Calendar component also support `disable`, `readOnly` and invalid which represents visual changes useful for the user.

### Disable

<div style={{width: '280px'}}>
  <Calendar defaultValue="today" disabled/>
</div>

```jsx
<Calendar defaultValue="today" disabled />
```

### Readonly

<div style={{width: '280px'}}>
  <Calendar defaultValue="today" readOnly onChange={e => console.log(e)} />
</div>

```jsx
<Calendar defaultValue="today" readOnly onChange={e => console.log(e)} />
```

## Min Max props

The calendar can be configured to allow only specific date range for selection.
Thus the dates with that range are enabled and others are disabled.

<div style={{width: '280px'}}>
  <Calendar min="2010-07-19" max="today" />
</div>

```jsx
<Calendar min="2010-07-19" max="today" />
```

## Handling Events

Events handlers like onChange and other user actions can be registered on a `Calendar`. A function that will be called when an date is selected.

<div style={{width: '280px'}}>
  <Calendar onChange={e => console.log(e)} />
</div>

```jsx
<Calendar onChange={e => console.log(e)} />
```
