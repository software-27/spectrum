## Overview

A **CycleButton** can be clicked repeatedly to change the state of something. Clicking the **CycleButton** advances to the next state in the cycle. For example, a play/pause button can be clicked once to initiate playback, then clicked again to pause the playback, and that cycle can be repeated ad infinitum.

The `actions` prop is required. It is an array composed of objects representing the actions for the CycleButton. Each object in the actions array should contain a `name`, `icon` and `label`.

## Example

<CycleButton
	onAction={(action, e) => console.log(action, e)}
	actions={[{name: 'play', icon: <PlayCircle />, label: 'Play'}, {name: 'pause', icon: <PauseCircle />, label: 'Pause'}]} />

```jsx
import CycleButton from '@react/react-spectrum/CycleButton';
import PlayCircle from '@react/react-spectrum/Icon/PlayCircle';
import PauseCircle from '@react/react-spectrum/Icon/PauseCircle';

<CycleButton
	onAction={(action, e) => console.log(action, e)}
	actions={[{name: 'play', icon: <PlayCircle />, label: 'Play'}, {name: 'pause', icon: <PauseCircle />, label: 'Pause'}]} />
```

## Action

Users can either let the component handle its own action (uncontrolled) or manage the action themselves (controlled). If no `defaultAction` or `action` is provided, the component will select the first element in the `actions` array as the default.

If you wish to use the component in a controlled state, an `action` string (matches action.name), an `onAction` method (passes the current action name in the list), and an `onChange` method (passes the next action name in the list) can be passed as props.

### defaultAction (uncontrolled)

<CycleButton
	onAction={(action, e) => console.log(action, e)}
	actions={[{name: 'play', icon: <PlayCircle />, label: 'Play'}, {name: 'pause', icon: <PauseCircle />, label: 'Pause'}]}
	defaultAction='pause' />

```jsx
import CycleButton from '@react/react-spectrum/CycleButton';
import PlayCircle from '@react/react-spectrum/Icon/PlayCircle';
import PauseCircle from '@react/react-spectrum/Icon/PauseCircle';

<CycleButton
	onAction={(action, e) => console.log(action, e)}
	onChange={(nextAction, e) => console.log(nextAction, e)}
	actions={[{name: 'play', icon: <PlayCircle />, label: 'Play'}, {name: 'pause', icon: <PauseCircle />, label: 'Pause'}]}
	defaultAction='pause' />
```

### Controlled Component
