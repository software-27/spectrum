import Dial from '@react/react-spectrum/Dial';

## Overview

A **Dial** allows a user to select a number value between a min/max. It's a slider, just looks a little different.

## Example
<Dial  renderLabel label="Label" />

```jsx
<Dial  renderLabel label="Label" />
```

## Size

<Dial size="S"  label="Size : s" />

```jsx
<Dial size="S" label="Size : s" />
```

<Dial label="Size : L" size="L" />

```jsx
<Dial label="Size : L" size="L" />
```

## State

Disabled will prevent user interaction and will grey out the component as a visual indication.

<Dial disabled/>

```jsx
<Dial disabled />
```
