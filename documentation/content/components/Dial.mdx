## Overview

A **Dial** allows a user to select a number value between a min/max. It's a slider, just looks a little different.

## Example

<Dial renderLabel label="Label" />

```jsx
<Dial renderLabel label="Label" />
```

## State

Disabled will prevent user interaction and will grey out the component as a visual indication.

<Dial disabled label="disabled" />

```jsx
<Dial disabled label="disabled" />
```
## Internationalization

To internationalize a dial, a localized string should be passed to the <code>label</code> prop.

## Accessibility

Dials are accessible by default, rendered in HTML using the `<input type="range">` element. It can receive keyboard focus and the value can be incremented using the UP/LEFT or decremented using the DOWN/RIGHT arrow keys.

### Labelling

Dial must have a label to be properly identified to users of assistive technology. The easiest way to satisfy this requirement is by adding a `label` prop. Setting `renderLabel` to `true` will display the label.

#### Using `label` prop

<p>
 <Dial size="L" label="Label" />
</p>
<p>
 <Dial size="L" label="Label" renderLabel />
</p>
