import Card from '@react/react-spectrum/Card';
import DocumentOutline from '@react/react-spectrum/Icon/DocumentOutline';

Cards are a common element used for browsing. A Card is an element used to display custom content, generally in the form of items a user can buy, own, read, or that a user has saved or created. Essentially cards are a set of THINGS, and the card is a window from the item to the viewer. It does not show all the possible information associated with the item, just enough to give the user an idea of what the card represents.

## Example

Here is an example of a Card that might be used to display some basic content. 

```jsx
<Card>
  <div>
    <div style={{'text-align': 'center'}}>
      <div style={{height: 136 + 'px', 'background-color': 'rgb(234, 234, 234)', 'border-top-left-radius': 4 + 'px', 'border-top-right-radius': 4 + 'px'}}>
        <DocumentOutline size="XXL" style={{'margin-top': 35 + 'px'}} />
      </div>
    </div>
    <div style={{'background-color': 'white', height: 120 + 'px', 'border-bottom-left-radius': 4 + 'px', 'border-bottom-right-radius': 4 + 'px'}}>
      <div style={{'display': 'flex', 'justify-content': 'space-between', 'padding-top': 15 + 'px', 'padding-right': 10 + 'px'}}>
        <div style={{'margin-left': 20 + 'px', 'color': 'rgb(75, 75, 75)'}}>Card Title</div>
      </div>
      <div style={{'margin-left': 20 + 'px', 'color': 'rgb(112, 112, 112)'}}>Optional Metadata
        <br /> Can be two to three lines.
      </div>
    </div>
  </div>
</Card>
```

## Internationalization

Internationalization of a Card is up to the application. Localized strings should be passed for the content of the Card.

## Accessibility

It is up to the application to ensure that the content of the card is accessible to the user. However, tab selection is available to the user, and the checkbox's functionality can be accessed through a popover title which displays on hover.
