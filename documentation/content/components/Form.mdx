## Overview

For building cohesive web forms with table alignment qualities.

## Examples

  <FormItem label="Life Story">

### Accessibility

Form is rendered using the HTML `form` element. If a label is provided by adding either the `aria-label` or `aria-labelledby` prop, the element will be identified to assistive technology as a form landmark region.

FormItem is rendered using an HTML `div` to wrap a `label` and the corresponding field. The `label` will automatically be associated to the field using the `for` attribute, and for composite controls that group related components into a group or fieldset, like a [ComboBox](/components/ComboBox), [Datepicker](/components/Datepicker), [NumberInput](/components/NumberInput), [RadioGroup](/components/RadioGroup), or [SearchWithin](/components/SearchWithin), by using the `aria-labelledby` prop.

When no `label` prop is provided, FormItem renders an empty `div` in place of the `label` so that the field content is not labeled by an empty element.
