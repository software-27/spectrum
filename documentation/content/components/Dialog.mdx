## Overview

An **Dialog** in Spectrum is modal window that forces the user to take some action before resuming to backgroud.

## Example

<ModalTrigger>
  <Button label="Open Dialog" variant="primary" />
  <Dialog title="Dialog title" confirmLabel="OK">
    <span>
    Content of the dialog
    </span>
  </Dialog>
</ModalTrigger>


```jsx
<ModalTrigger>
  <Button label="Open Dialog" variant="primary" />
  <Dialog  title="Dialog title" confirmLabel="OK">
    <span>
    Content of the dialog
    </span>
  </Dialog>
</ModalTrigger>
```

## Buttons

Spectrum 'Dialog' allows confirm and cancel button to dismiss the dialog accordingly.

<ModalTrigger>
  <Button label="Open Dialog" variant="primary" />
  <Dialog
    title="Dialog title"
    confirmLabel="OK"
    cancelLabel="Cancel" >
    <span>
    Content of the dialog
    </span>
  </Dialog>
</ModalTrigger>

```jsx
<ModalTrigger>
  <Button label="Open Dialog" variant="primary" />
  <Dialog
    onClose={close}
    title="Dialog title"
    confirmLabel="OK"
    cancelLabel="Cancel" >
    <span>
    Content of the dialog
    </span>
  </Dialog>
</ModalTrigger>
```

## Keyboard Confirm

Setting keyboardConfirm prop to true makes it so pressing the Enter key executes the default action for the Dialog. It is the equivalent of pressing the “Confirm” button.

<ModalTrigger>
  <Button label="Open Dialog" variant="primary" />
  <Dialog
    onClose={close}
    title="Dialog Title"
    backdropClickable
    confirmLabel="OK"
    cancelLabel="Cancel"
    keyboardConfirm
    onConfirm={e => console.log(e)}>
    <Textfield aria-label="Textfield" placeholder="Textfield" autoFocus />
  </Dialog>
</ModalTrigger>

```jsx
<ModalTrigger>
  <Button label="Open Dialog" variant="primary" />
  <Dialog
    onClose={close}
    title="Dialog Title"
    backdropClickable
    confirmLabel="OK"
    cancelLabel="Cancel"
    keyboardConfirm
    onConfirm={e => console.log(e)}>
    <Textfield aria-label="Textfield" placeholder="Textfield" autoFocus />
  </Dialog>
</ModalTrigger>
```
