import Button from '@react/react-spectrum/Button';
import VisuallyHidden from '@react/react-spectrum/VisuallyHidden';

## Overview

A **VisuallyHidden** component provides text that is visually hidden yet remains accessible to assistive technology like screen readers.

This is useful for providing additional context when labelling a control, which may be necessary to satisfy [WCAG 2.1 Success Criterion 2.4.4 Link Purpose (In Context)](https://www.w3.org/TR/WCAG21/#link-purpose-in-context).

## Example

<Button
  variant="cta"
  element="a"
  href="https://commerce.adobe.com/checkout?cli=adobe_com&co=US&lang=en&items[0][id]=632B3ADD940A7FBB7864AA5AD19B8D28&items[0][cs]=0"
  target="_blank">
  Buy <VisuallyHidden>Creative Cloud</VisuallyHidden> now
</Button>

```jsx
import Button from '@react/react-spectrum/Button';
import VisuallyHidden from '@react/react-spectrum/VisuallyHidden';

<Button
  variant="cta"

  target="_blank">
  Buy <VisuallyHidden>Creative Cloud</VisuallyHidden> now
</Button>
```

## Variants

### element

Use the `element` prop to specify the DOM element used to contain the visually hidden content.

<VisuallyHidden element="div">A visually hidden div element</VisuallyHidden>

```jsx
<VisuallyHidden element="div">A visually hidden div element</VisuallyHidden>
```

### focusable

The `focusable` boolean property is useful for implementations such as "skip links" where a visually hidden interactive control should become visible when it receives focus.

<VisuallyHidden element="a" href="#main" className="spectrum-Link" focusable>Skip to Main Content</VisuallyHidden>&nbsp;

```jsx
<VisuallyHidden element="a" href="#main" className="spectrum-Link" focusable>Skip to Main Content</VisuallyHidden>&nbsp;
```
