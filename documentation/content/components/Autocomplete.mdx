## Overview
An Autocomplete relies on a method, `getCompletions`, to predict the string a user will want. `getCompletions` can implement
any kind of predictive algorithm, could be based on past strings, could be a predefined list. A user must select a string
from the list by default. Using allowCreate we can allow the user to enter any string, regardless of its existence in
the list.

### Default

  <Autocomplete getCompletions={() => ['Chocolate', 'Vanilla']}>
    <Textfield placeholder="Autocomplete..." />
  </Autocomplete>

```jsx
 <Autocomplete getCompletions={() => ['Chocolate', 'Vanilla']}>
   <Textfield placeholder="Autocomplete..." />
 </Autocomplete>
```


## Autocomplete with allowCreate
Try typing 'puppy' and hitting enter on the above example and this one.

  <Autocomplete allowCreate getCompletions={() => ['Chocolate', 'Vanilla']}>
   <Textfield placeholder="Autocomplete..." />
  </Autocomplete>


```jsx
  <Autocomplete allowCreate getCompletions={() => ['Chocolate', 'Vanilla']}>
    <Textfield placeholder="Autocomplete..." />
  </Autocomplete>
```

## Accessibility

### WAI-ARIA Design Pattern

The Autocomplete component implements the ComboBox design pattern, per [WAI-ARIA 1.1](https://www.w3.org/TR/wai-aria-practices/#combobox).

### Labelling

An Autocomplete component must be labelled for Accessibility using either a FieldLabel, or either the `aria-label` or `aria-labelledby` attribute.

#### Implicit labelling
  <FieldLabel label="Ice Cream Flavors">
    <Autocomplete
      getCompletions={() => ['Chocolate', 'Vanilla']}>
      <Textfield placeholder="Autocomplete..." />
    </Autocomplete>
  </FieldLabel>

```jsx
  <FieldLabel label="Ice Cream Flavors">
    <Autocomplete
      getCompletions={() => ['Chocolate', 'Vanilla']}>
      <Textfield placeholder="Autocomplete..." />
    </Autocomplete>
  </FieldLabel>
```
#### Explicit labelling
  <FieldLabel label="Ice Cream Flavors" labelFor="autocomplete-id" />
  <Autocomplete
    getCompletions={() => ['Chocolate', 'Vanilla']}
    id="autocomplete-id">
    <Textfield placeholder="Autocomplete..." />
  </Autocomplete>

```jsx
  <FieldLabel label="Ice Cream Flavors"  labelFor="autocomplete-id" />
  <Autocomplete
    getCompletions={() => ['Chocolate', 'Vanilla']}
    id="autocomplete-id">
    <Textfield placeholder="Autocomplete..." />
  </Autocomplete>
```
