## Overview

Sliders allow users to quickly and efficiently select a value within a predefined
range of values. Therefore sliders should be used in cases where there are upper
and lower bounds to the selection.

## Example

### Basic slider

<Slider label="Basic" />

```jsx
<Slider />
```

### Variant

#### Ramp

<Slider label="Ramp" variant="ramp" />

```jsx
<Slider variant="ramp" />
```

The variant `ramp` gives you a slightly different look and feel on the slider.

#### Range

<Slider label="range" variant="range" />

```jsx
<Slider variant="range" />
```

### Slider with step

<Slider label="With step" step={10} />

```jsx
<Slider step={10} />
```

`Step` allows you set the incremental or decremental step value.

### Filled slider

<Slider label="Filled" filled />

```jsx
<Slider filled />
```

Setting `filled` to true highlights the line from the `min` value
to the selected value.

Setting `filled` to true along with some value in `fillOffset` highlights the line from the `fillOffset` value

## Accessibility

Slider has been implemented using the [HTML5 `input` control with `type=range`](https://www.w3.org/TR/html52/sec-forms.html#range-state-typerange). This allows gestures for incrementing or decrementing the Slider value to work with touch screen readers, like VoiceOver on iOS, TalkBack on Android or Narrator on Windows.

The `range` variant behaves as a fieldset containing two `input` controls with `type=range`.

### Labelling

A Slider can be labelled using the `label` prop which will be rendered if `renderLabel` is `true`, or will behave as an `aria-label` when `renderLabel` is `false`.

<Slider renderLabel label="label"  variant="range" defaultStartValue={33} defaultEndValue={66} />

```jsx
<Slider renderLabel label="label"  variant="range" defaultStartValue={33} defaultEndValue={66} />
```

A Slider can also be labelled using a FieldLabel and `aria-labelledby`.

<FieldLabel label="label" labelFor="slider-id" id="slider-label-id" />
<Slider id="slider-id" aria-labelledby="slider-label-id" variant="range" defaultStartValue={33} defaultEndValue={66} />

```jsx
<FieldLabel label="label" labelFor="slider-id" id="slider-label-id" />
<Slider id="slider-id" aria-labelledby="slider-label-id" variant="range" defaultStartValue={33} defaultEndValue={66} />
```

### getAriaValueText

Use the `getAriaValueText` prop to specify a function to return a string to use as the `aria-valuetext` for a Slider `input` from its `value`.

<Slider
  min={0}
  max={1440}
  defaultStartValue={600}
  defaultEndValue={840}
  variant="range"
  renderLabel
  label="Duration"
  getAriaValueText={
    minutes => {
      const date = new Date();
      date.setHours(Math.floor(minutes / 60));
      date.setMinutes(minutes % 60);
      return date.toLocaleTimeString('en-us', {hour: '2-digit', minute: '2-digit'});
    }
  } />
