import Label from '@react/react-spectrum/Label';

## Overview

A **Label** displays a string inside a rounded rectangle with a status color background.

## Example

<Label>Hello</Label>

### Variant

<Label variant="grey">Grey</Label>
<Label variant="green">Green</Label>
<Label variant="blue">Blue</Label>
<Label variant="red">Red</Label>
<Label variant="orange">Orange</Label>
<Label variant="and">And</Label>
<Label variant="or">Or</Label>
<Label variant="active">Active</Label>
<Label variant="inactive">Inactive</Label>

```jsx

```

### Size

In addition to the standard sized label, a large version is available by specifying the `size` prop
with a value of `L`.

<Label size="L">Large</Label>
