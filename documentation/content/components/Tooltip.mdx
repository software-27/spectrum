import Tooltip from '@react/react-spectrum/Tooltip';

## Overview

Tooltips appear when a user hovers, taps, or focuses on an element.
The information it displays is simple and concise. It can be the description
of an icon, a simple explanation of a behavior or help text.

## Example

<Tooltip>Hello World</Tooltip>


## Variants

<Tooltip variant="info">info</Tooltip>
<Tooltip variant="inspect">inspect</Tooltip>
<Tooltip variant="success">success</Tooltip>
<Tooltip variant="error">error</Tooltip>

```jsx
import Tooltip from '@react/react-spectrum/Tooltip';

<Tooltip variant="info">Info</Tooltip>
<Tooltip variant="inspect">Info</Tooltip>
<Tooltip variant="success">Info</Tooltip>
<Tooltip variant="error">Info</Tooltip>
```
