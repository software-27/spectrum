## Overview

Tooltips appear when a user hovers, taps, or focuses on an element.
The information it displays is simple and concise. It can be the description
of an icon, a simple explanation of a behavior or help text.

## Example

<Tooltip>Hello World</Tooltip>

## Accessibility

### WAI-ARIA design pattern

Tooltip, when used together with the [OverlayTrigger](OverlayTrigger) component, implements the [WAI-ARIA 1.1 Tooltip Widget](https://www.w3.org/TR/wai-aria-practices-1.1/#tooltip) design pattern.  

When an OverlayTrigger opens an element with `role` equal to `tooltip`, an appropriate `aria-describedby` attribute will be added to the trigger element.

<OverlayTrigger placement="right">
  <Button label="Tooltip" variant="cta" />
  <Tooltip variant="info">
    Displays a tooltip
  </Tooltip>
</OverlayTrigger>

```jsx
<OverlayTrigger placement="right">
```
