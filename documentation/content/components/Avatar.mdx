## Overview

An **Avatar** is used to show a user's profile picture or other small avatar.

## Internationalization

To internationalize an avatar, a localized string should be passed to the `alt` prop of the avatar.

## Accessibility

An Avatar component renders using an `img` tag. A localized string should be passed to the `alt` prop of the Avatar to provide alternative text to the `img` using the `alt` attribute. If the avatar is used purely for decoration, and alternative text describing the avatar would be redundant, not assigning an `alt` prop will cause `img` to be rendered using an empty `alt=""` attribute.
