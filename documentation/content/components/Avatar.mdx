## Overview

An **Avatar** is used to show a user's profile picture or other small avatar.

## Example

### Default

<Avatar src="http://spectrum-css.corp.adobe.com/2.0.0-beta.91/docs/img/example-ava.jpg" alt="User" />

```jsx
<Avatar src="http://spectrum-css.corp.adobe.com/2.0.0-beta.91/docs/img/example-ava.jpg" alt="User" />
```

### Disabled

<Avatar src="http://spectrum-css.corp.adobe.com/2.0.0-beta.91/docs/img/example-ava.jpg" alt="User" disabled />


```jsx
<Avatar src="http://spectrum-css.corp.adobe.com/2.0.0-beta.91/docs/img/example-ava.jpg" alt="User" disabled />
```

## Internationalization

To internationalize an avatar, a localized string should be passed to the `alt` prop of the avatar.

## Accessibility

Avatars are accessible by default. The correct aria roles will automatically be applied.
