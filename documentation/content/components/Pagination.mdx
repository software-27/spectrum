import Pagination from '@react/react-spectrum/Pagination';

## Overview

Pagination tracks a current index between 1 and the totalPages. It emits events for progressing by 1 forward or backward.


## Example

<Pagination />

```jsx
<Pagination />
```

## Accessibility

By default, the Pagination control is identified as a navigation region labelled for accessibility as "Page navigation".
This label can be overridden using either `aria-label` or `aria-labelledby`.

The previous and next buttons are labelled as "Previous" and "Next" respectively.

The text field for explicitly entering the current page has the default label "Page".

<Pagination aria-label="Table pagination" variant="explicit" mode="primary" totalPages={5} defaultPage={3} />

```jsx
<Pagination aria-label="Table pagination" variant="explicit" mode="primary" totalPages={5} defaultPage={3} />
```
