import Clock from '@react/react-spectrum/Clock';

## Overview

The **Textfield** provides an input element for users to enter in data (for forms and such). It is restricted to a single line of text.

## Examples

<Textfield placeholder="Text here" aria-label="Text field" />

```jsx
<Textfield placeholder="Text here" />
```

## Variants

### Quiet

<Textfield quiet placeholder="Quiet variant" aria-label="Quiet variant" />

```jsx
<Textfield quiet placeholder="Quiet variant" />
```

### Disabled

Disables text field from interaction

<Textfield disabled placeholder="Disabled variant" aria-label="Disabled variant" />

```jsx
<Textfield disabled placeholder="Disabled variant" />
```

### Validation State

There are two validation states: valid, and invalid.

<Textfield validationState="valid" placeholder="Valid" style={{marginRight: 20}} aria-label="Valid state" />
<Textfield validationState="invalid" placeholder="Invalid" aria-label="Invalid state" />

```jsx
<Textfield validationState="valid" placeholder="Valid" />
<Textfield validationState="invalid" placeholder="Invalid" />
```

### Read-only

Allows text to be selected but not changed

<Textfield readOnly value="Read-only variant" aria-label="Read-only variant" />

```jsx
<Textfield readOnly value="Read-only variant" />
```

### Required

Shows warning if text field is empty

<Textfield required placeholder="Required variant" aria-label="Required variant" />

```jsx
<Textfield required placeholder="Required variant" />
```

### MultiLine

Allows user to input multiple lines of text

<Textfield multiLine aria-label="Multiline variant" />

```jsx
<Textfield multiLine />
```

## Events

### onChange

Function fired when value of textfield is changed.

<Textfield onChange={e => console.log(e)} aria-label="Textfield with onChange event handler" />

```jsx
<Textfield onChange={e => console.log(e)} />
```

### onFocus

Function fired when the textfield is focused.

<Textfield onFocus={e => console.log(e)} aria-label="Textfield with onFocus event handler" />

```jsx
<Textfield onFocus={e => console.log(e)} />
```

### onBlur

Function fired when the focus on the textfield is lost.

<Textfield onBlur={e => console.log(e)} aria-label="Textfield with onBlur event handler" />

```jsx
<Textfield onBlur={e => console.log(e)} />
```

## Accessibility

Textfield has been implemented using the [HTML5 `input` control with `type=text`](https://www.w3.org/TR/html52/sec-forms.html#text-typetext-state-and-search-state-typesearch), and provided it is labelled appropriately, is be accessible by default. 

### Labelling

A Textfield must be labelled using a [FieldLabel](/components/FieldLabel), `aria-label` or `aria-labelledby`, to provide context for Accessibility.

#### Implicit labelling with `FieldLabel`

<FieldLabel label="First name">
  <Textfield />
</FieldLabel>

```jsx
<FieldLabel label="First name">
  <Textfield />
</FieldLabel>
```

#### Explicit labelling using `FieldLabel` and `labelFor` property

A FieldLabel can explicitly label a Textfield using the `labelFor` prop.

<FieldLabel label="First name" labelFor="textfield-id" />
<Textfield id="textfield-id" />

```jsx
<FieldLabel label="First name" labelFor="textfield-id" />
<Textfield id="textfield-id" />
```

#### Using `aria-label` and `aria-labelledby`

Use `aria-label` and `aria-labelledby` in rare cases, like the [Clock](Clock) component, where the TextField does not have a visible label or where the Textfield should be labelled by an external element.

<FieldLabel label="Clock" labelFor="clock-id" id="clock-id-label" />
<Clock id="clock-id" aria-labelledby="clock-id-label" value="10:30" />

```jsx
<FieldLabel label="Clock" labelFor="clock-id" id="clock-id-label" />
<div
  class="react-spectrum-Clock"
  role="group"
  id="clock-id-group"
  aria-labelledby="clock-id-label clock-id-time-label">
  <time
    class="u-react-spectrum-screenReaderOnly"
    id="clock-id-time-label"
    aria-live="off"
    hidden>10:30</time>
  <Textfield
    type="number"
    min="0"
    max="23"
    class="react-spectrum-Clock-hour"
    value="10"
    placeholder="HH"
    id="clock-id"
    aria-label="Hours"
    aria-labelledby="clock-id-label clock-id" />
  <span class="react-spectrum-Clock-divider">:</span>
  <Textfield
    type="number"
    min="0"
    max="59"
    class="spectrum-Textfield react-spectrum-Clock-minute"
    value="30"
    placeholder="mm"
    id="clock-id-minutes"
    aria-label="Minutes"
    aria-labelledby="clock-id-label clock-id-minutes" />
</div>
```
