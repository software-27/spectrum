import AdobeExperienceManagerColorLight from '@react/react-spectrum/Icon/AdobeExperienceManagerColorLight';
import Bell from '@react/react-spectrum/Icon/Bell';
import Brush from '@react/react-spectrum/Icon/Brush';
import Icon from '@react/react-spectrum/Icon';

## Overview

**Icons** can easily be used via importing them from `react-spectrum/Icon/IconName` and rendering as `<IconName />`.

## Example

<Bell/>
<Brush/>

```jsx
<Bell/>
<Brush/>
```

## Variants

Icons are available in various sizes in Spectrum ranging from `XXS` to `XXL`, `M` being the default. We can specify the size via 'size' prop.

### Size variants

<Bell size="XS" />
<Bell size="S" />
<Bell size="M" />
<Bell size="L" />
<Bell size="XL" />

```jsx
<Bell size="XS" />
<Bell size="S" />
<Bell size="M" />
<Bell size="L" />
<Bell size="XL" />
```

In addition to 'size' prop, Spectrum supports `Color icon` and `Custom SVG` as well.

### Color Icon

<AdobeExperienceManagerColorLight size="XL" alt="Adobe Experience Manager" />

```jsx
<AdobeExperienceManagerColorLight size="XL" alt="Adobe Experience Manager" />
```

### Custom SVG

<Icon>
<svg viewBox="0 0 25 25">
<rect
  x="0"
  y="0"
  width="25"
  height="25"
/>
</svg>
</Icon>

```jsx
<Icon>
  <svg viewBox="0 0 25 25">
    <rect
      x="0"
      y="0"
      width="25"
      height="25"
    />
  </svg>
</Icon>
```

## Accessibility

`aria-hidden` is set to `true` by default for Icons. The `alt` prop suppresses this attribute and adds the alt text as the aria-label of the icon

<AdobeExperienceManagerColorLight size="XL" alt="Adobe Experience Manager" />

```jsx
<AdobeExperienceManagerColorLight size="XL" alt="Adobe Experience Manager" />
```
