import FieldLabel from '@react/react-spectrum/FieldLabel';
import Radio from '@react/react-spectrum/Radio';
import RadioGroup from '@react/react-spectrum/RadioGroup';


## Overview

A **Radio button** allow users to select/deselect a particular option. It can be checked or unchecked depending on the user needs.

## Example

<Radio label="React" />

```jsx
<Radio label="React" />
```

## Content

Radio Buttons can have a label or not. A label describes what the button represents, and tell users what to do. This is important for accessibility, since a screen reader will read the label before each option. A label can be provided to the radio button through the `label` prop.

### Label Set

<Radio label="React" />

```jsx
<Radio label="React" />
```

### Label Not Set

<Radio label="" aria-label="React" />

```jsx
<Radio label="" aria-label="React" />
```

## Variants

### Standard Radio Buttons

Standard radio buttons are the default style for radio buttons. The blue color provides a visual prominence that is optimal for forms, settings, etc. where the radio buttons need to be noticed.

<Radio label="React" defaultChecked />

```jsx
<Radio label="React" defaultChecked />
```

### Quiet Radio Buttons

Standard checkboxes are the default style for checkboxes. The blue color provides a visual prominence that is optimal for forms, settings, lists or grids of assets, etc. where the checkboxes need to be noticed.

<Radio label="React" defaultChecked quiet />

```jsx
<Radio label="React" defaultChecked quiet />
```

### Alignment

#### Labels Below

The button label can either be adjacent (default) or below the button. The latter can be achieved via the 'labelBelow' prop.

<Radio label="React" labelBelow />

```jsx
<Radio label="React" labelBelow />
```

## States

Radio buttons have a number of props for controlling their visual/checked state. The `disabled` prop, which applies a disabled style to the button, and also prevents clicks from activating it. The 'checked' prop is also available to apply a checked appearance to the radio button. In addition to this 'defaultChecked' and 'invalid' prop is also provided.

<Radio label="React" disabled />
<Radio label="React" checked={false} />
<Radio label="React" checked />
<Radio label="React" invalid />
<Radio label="React" defaultChecked />

```jsx
<Radio label="React" disabled />
<Radio label="React" checked={false} />
<Radio label="React" checked />
<Radio label="React" invalid />
<Radio label="React" defaultChecked />
```

## Handling Events

Events handlers for change can be registered on a `Radio Button` as shown below.

<Radio label="ClickMe" onChange={e => alert('Radio Button clicked!')} />

```jsx
<Radio label="ClickMe" onChange={e => alert('Radio Button clicked!')} />
```

## Internationalization

To internationalize a radio button, a localized string should be passed to the `label` prop of the radio button.

## Accessibility

Radios are accessible by default, rendered in HTML using the `<input type="radio">` element. When the radio is `checked`, the appropriate ARIA state attribute will automatically be applied.

### Labelling

A Radio must have a label to be properly identified to users of assistive technology. The easiest way to satisfy this requirement is by adding a `label` prop or by including the label text as child content.

<div><code>label</code> prop: </div><Radio label="React" />

<div>as content: </div><Radio defaultChecked>React</Radio>

<div>not the best practice: </div><Radio renderLabel={false} label="React radio" defaultChecked>with renderLabel: false</Radio>

```jsx
<div><code>label</code> prop: </div><Radio label="React" />

<div>as content: </div><Radio defaultChecked>React</Radio>

<div>not the best practice: </div><Radio renderLabel={false} label="React radio" defaultChecked>with renderLabel: false</Radio>
```

#### RadioGroup

`RadioGroup` should be used to label a group of related `Radio` as a fieldset. The RadioGroup label can be provided using `aria-label` or `aria-labelledby`.

<FieldLabel label="Love me some React!" id="radiogroup-label" />
<RadioGroup name="radiogroup" aria-labelledby="radiogroup-label" vertical>
  <Radio label="Yes" value="Yes" />
  <Radio label="No" value="No" />
</RadioGroup>

```jsx
<FieldLabel label="Love me some React!" id="radiogroup-label" />
<RadioGroup name="radiogroup" aria-labelledby="radiogroup-label" vertical>
  <Radio label="Yes" value="Yes" />
  <Radio label="No" value="No" />
</RadioGroup>
```
